

# نظام إدارة الخطة الاستثمارية - هيئة ميناء دمياط
## Investment Plan Management System

### نظرة عامة
نظام ERP متكامل باللغة العربية (RTL) لإدارة الخطة الاستثمارية لهيئة ميناء دمياط، يشمل 20 شاشة مع خريطة GIS ونظام صلاحيات متكامل.

### التقنيات
- **Frontend**: React + Tailwind CSS + Recharts + Leaflet (للخرائط)
- **Backend**: Supabase (PostgreSQL + Auth + RLS)
- **اللغة**: عربي بالكامل مع دعم RTL

---

### 1. قاعدة البيانات (Supabase)

**جداول أساسية (~20 جدول):**
- `departments` - الإدارات
- `profiles` - بيانات المستخدمين
- `user_roles` + enum `app_role` (admin, manager, viewer) - الأدوار والصلاحيات
- `investment_operations` - العمليات الاستثمارية المقترحة
- `project_studies` - دراسات المشاريع
- `strategic_plans` - الخطة الاستراتيجية المعتمدة
- `operation_history` + `operation_status` - تتبع حالة العمليات
- `execution_tracking` + `milestones` - متابعة التنفيذ
- `risk_register` - سجل المخاطر
- `tenders` + `tender_bidders` - المناقصات
- `contracts` + `contractors` - العقود والمقاولين
- `budgets` + `financial_sources` - الميزانيات
- `payments` + `invoices` - المدفوعات والمستخلصات
- `project_locations` + `gis_layers` - بيانات الخريطة
- `attachments` - المرفقات
- `notifications` - الإشعارات
- `audit_log` - سجل العمليات

**RLS**: صلاحيات حسب الدور باستخدام `has_role()` security definer function.

---

### 2. نظام المصادقة والصلاحيات
- تسجيل دخول بـ Email + Password عبر Supabase Auth
- 3 أدوار: **مدير النظام** / **مدير إدارة** / **مستخدم عادي**
- صفحة إدارة المستخدمين والصلاحيات

---

### 3. الشاشات (20 شاشة)

**التخطيط العام**: Sidebar عربي (يمين) + محتوى رئيسي + Header مع إشعارات

#### المجموعة 1 - الأساسيات
1. **تسجيل الدخول** - نموذج بسيط مع شعار الميناء
2. **لوحة التحكم** - KPIs (إجمالي العمليات، الميزانية، نسبة التنفيذ، المصروفات) + رسوم بيانية Recharts
3. **إدارة المستخدمين** - جدول + إضافة/تعديل/حذف
4. **إدارة الصلاحيات** - تعيين أدوار للمستخدمين

#### المجموعة 2 - العمليات الاستثمارية
5. **العمليات المقترحة** - جدول مع فلترة وبحث
6. **إضافة عملية** - نموذج (اسم، إدارة، قيمة تقديرية، مدة)
7. **دراسة المشروع** - فكرة، مبررات، وصف فني، تحليل
8. **اعتماد الخطة** - موافقة/رفض/تعديل ميزانية
9. **الخطة الاستثمارية** - العمليات المعتمدة في جدول شامل

#### المجموعة 3 - التنفيذ والمتابعة
10. **متابعة التنفيذ** - نسبة إنجاز + إنجازات + معوقات
11. **إدارة المشتريات** - مناقصات + ترسية + عقود
12. **العقود** - قيمة العقد + المقاول + التواريخ

#### المجموعة 4 - المالية
13. **الإدارة المالية** - مستخلصات ومدفوعات
14. **إضافة مستخلص** - رقم، مبلغ، تاريخ، مرفقات

#### المجموعة 5 - GIS والتقارير
15. **خريطة GIS** - Leaflet map centered on (31.4666, 31.7500) مع طبقات المشاريع
16. **تقارير التنفيذ** - رسوم بيانية للإنجاز
17. **تقارير الميزانية** - Budget vs Spending
18. **تقارير الإدارات** - مشاريع حسب الإدارة

#### المجموعة 6 - النظام
19. **سجل العمليات** - Audit Log
20. **إعدادات النظام** - إعدادات عامة

---

### 4. خريطة GIS
- مكتبة **Leaflet** مع OpenStreetMap tiles
- مركز الخريطة: ميناء دمياط (31.4666, 31.7500)
- طبقات: حدود الميناء، المحطات، المشاريع الاستثمارية
- علامات تفاعلية لكل مشروع مع popup يعرض التفاصيل

---

### 5. التصميم
- **نمط**: لوحة تحكم حكومية حديثة (مشابه لـ SAP)
- **ألوان**: أزرق داكن + أبيض + رمادي (ألوان رسمية)
- **RTL**: كامل من اليمين لليسار
- **خط**: عربي واضح
- **Responsive**: يعمل على الشاشات الكبيرة والأجهزة اللوحية
